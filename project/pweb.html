+ what is web development?

does web-development means just creating website?

the answer is no. the word web development refers to any system architecture that can run on the web environment. think of this as software, instead of running on the computer, it runs on servers which enables you to use the application using your browsers. some examples: online banking application, tax-file application, NetFlix, GitHub etc.
