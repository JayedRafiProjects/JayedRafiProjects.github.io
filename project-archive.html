Project Title: One Scan - Business Card Generation & Process Improvement (Web App)
Status / Duration: January 2022
Description: Around 15 billion trees are cut down every year. Surprisingly the majority percentage of it is for making papers. My initial approach behind working on this project was to reduce this number. This project is a web-based system that helps user create their QR-readable business card. This platform contains an easy walk-through of creating a QR-readable business card from scratch. It has a design library containing multiple creative designs. I made the whole system open-source so anyone can easily add their design to the library. Another interesting part is the domain validator. It is a script using AJAX. This script checks the already taken folder name on the given server and tells the user about the availability. Also, I tried to make the user interface with a modern look.
Tools: JavaScript, AJAX, HTML, CSS, Bootstrap

Project Title: Chart Me - Data Visualization Application (Web App)
Status / Duration: November 2021
Description: This project is based on data visualization but not limited to numeric values. It is a web-based system and offers 5 different visualization formats (line chart, bar chart, doughnut chart, pie chart & scatter plot). It collects information from the user and evaluates based on Chart.js. Fun fact, while developing it, I used Durstenfeld Shuffle Algorithm to randomize chart colours (pie chart & doughnut chart)on each refresh. Unlike my other projects, this was programmed based on practical implementation (I was planning to use this for making a PowerPoint presentation at work) rather than introducing a new computational algorithm.
Tools: JavaScript, HTML, CSS, Chart.js

Project Title: Sorting Algorithm Visualizer (Terminal/Command)
Status / Duration: September 2021
Description: Though the name seems familiar, this project is unique. Unlike most other sorting algorithm visualizers, this project does not have any web or desktop interface. Its interface is the output interface of users' Java compiler. From my experience, an advanced text editor like Intellij Idea requires package installation for using additional libraries. It can be a huge obstacle for a student who recently started learning computer science. It helps students to compile and run this application as usual Java files without any package/library installation. Another fascinating part of this project is its calculation algorithm. This algorithm creates a terminal graphic using multi-dimensional arrays.
Tools: Java
